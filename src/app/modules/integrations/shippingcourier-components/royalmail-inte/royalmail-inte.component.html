Getting started


  1. Sign up on the API (Developer) Portal Opens in a new window

  2. Register your application: Once you've chosen an API, you'll need to register your application in the 'My Apps' section of your account. When you register, your application is assigned a unique client ID and client secret. These will be used to identify your application when you want to integrate to one of our APIs.
    • After Login intro API portal need to click on “MyApps” in header section.
    • In My Apps Page click on “Register Your Application” link or “Create new app button”
    • In Register Application page:
      • Enter the Application Name as ProShip(10 digit Royal Mail Account number) . Ex: ProShip1234567890
      • Enter the Description as Book Shipments through API and print labels

  3. Subscribe to a Royal Mail API: After you've registered your application, simply select and subscribe to the API you want. Some APIs are subject to approval; this usually takes less than a week. Then you're all set to integrate the API with your app, website or business system.
  4. Request approval: If you would like to subscribe to API Shipping, Tracking API or the Local Collect API please contact your Royal Mail account manager who will initiate your API integration. If you do not have a Royal Mail Account Manager, please contact the Business Helpdesk on 08457 950 950 or email:business.customer.service@royalmail.com
